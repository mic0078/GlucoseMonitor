💉 What is this?

Glucose Monitor is a lightweight Windows desktop application that fetches and displays real-time blood glucose readings from a FreeStyle Libre sensor (Abbott) via the LibreView / LibreLinkUp service. It runs quietly in the background and stays visible on your screen at all times.

📡

Live data

Readings every 60 seconds from the LibreView API. Automatic re-login when the session expires.

📊

History chart

Graphical glucose trend for the last ~8 hours with gridlines and time labels.

🔒

Secure password

Login credentials encrypted with Windows DPAPI — readable only by your Windows account.

📌

Always on top

Window stays visible above all other applications. Mini-widget mode 120×50 px.

🔄

Unit selection

One click switches between mmol/L and mg/dL. Chart, stats and tray tooltip update instantly.

🚀 Installation

1

Run the installer as Administrator

Right-click Install_GlucoseMonitor.bat and choose "Run as administrator". Required to set system permissions.

2

Installer copies files to C:\Glucose\

Scripts, icon and config file are placed in a dedicated folder with correct write permissions.

3

Creates a desktop shortcut

"Glucose Monitor" shortcut with the blood-drop icon. Launching it hides the PowerShell console — the app runs in the background.

4

First launch — log in

Enter the email and password from your LibreView account. Credentials will be encrypted and saved. On subsequent launches, login is automatic.

Tip: After installation you can add the shortcut to Startup (Win + R → shell:startup) so the app launches automatically with Windows.

👤 Switching accounts — monitoring another person

The app can be switched to a different LibreView account at any time — for example if you want to monitor readings from another person who has a FreeStyle Libre sensor (child, partner, patient). Each account is linked to a specific sensor and patient.

🖱️
Step 1
Right-click the blood-drop icon in the system tray (taskbar, bottom-right corner of the screen)
📋
Step 2
From the context menu choose "Switch account" — the app immediately logs out the current user and deletes the saved config file
🔑
Step 3
A login window appears — enter the new email and password from the LibreView account linked to the other sensor
✅
Step 4
After logging in, the app fetches readings from the new sensor. The password is encrypted and saved — next time, login is automatic
What happens under the hood:
The app deletes config.ini (email + encrypted password) and clears the API session token. After entering new credentials, login goes to api-eu.libreview.io (or the appropriate regional server), the new data is encrypted by Windows DPAPI and saved to config.ini. No restart required — the app immediately shows data from the new account.

🌍 Countries & regional availability

Glucose Monitor works in every country where the FreeStyle Libre sensor is officially available and linked to a LibreLinkUp account. No additional configuration is needed — the app automatically selects the correct API server for your region.

🔄 Automatic region detection:
On login, the app first connects to the default European server (api-eu.libreview.io). If your account belongs to a different region, the server returns a redirect response — the app instantly switches to the correct server and retries the login. The whole process takes a fraction of a second and is invisible to you.
